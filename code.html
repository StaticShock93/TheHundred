The Code for THE HUNDRED.


                        // get the values from the Page
                        // starts or controller function
                        function getValues() {
                            // get values from the page
                            let startValue = document.getElementById("startValue").value;
                            let endValue = document.getElementById("endValue").value;
                            // we need to validate our input
                            // parse into Integers and only accept #'s from 0-100
                            startValue = Math.min(Math.max(parseInt(startValue), 0), 100);
                            endValue = Math.min(Math.max(parseInt(endValue), 0), 100);
                        
                            if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                                // we call generateNumbers
                                let numbers = generateNumbers(startValue, endValue);
                                // we call displayNumbers
                                displayNumbers(numbers);
                            } else {
                                alert("You must enter integers only");
                            }
                        }
                        
                        
                        // generate numbers from startValue to the endValue 0-100
                        // logic function 1
                        function generateNumbers(sValue, eValue) {
                        
                            let numbers = [];
                            // we want to get all numbers from start to end
                            for (let index = sValue; index <= eValue; index++) {
                        
                                // this will execute in a loop until index = eValue
                                numbers.push(index);
                            }
                            return numbers;
                        }
                        
                        // we want to only allow for input values from 0-100
                        // logic function 2
                        function displayAlert() {
                            let startValue = document.getElementById("startValue").value;
                            let endValue = document.getElementById("endValue").value;
                        
                            if (startValue < 0 || startValue > 100) {
                                this.value = 0;
                                alert("You must enter a number from 0-100");
                            }
                            if (endValue < 0 || endValue > 100) {
                                this.value = 100;
                                alert("You must enter a number from 0-100");
                            }
                        }
                        
                        // display the numbers and mark even numbers bold
                        // display or view function
                        function displayNumbers(numbers) {
                        
                            let templateRows = "";
                        
                            for (let index = 0; index < numbers.length; index++) {
                        
                                let className = "even";
                                let number = numbers[index];
                        
                                if (number % 2 == 0) {
                                    className = "even";
                                } else {
                                    className = "odd";
                                }
                                //This does not render correctly with prism, see the source.
                                templateRows += `${number}`;
                        
                            }
                            document.getElementById("results").innerHTML = templateRows;
                        
                        }

The Code is structured in four functions.

GetValues

GetValues is the main, or controller, function of the application. This is the function that is called and executed upon clicking the "display numbers" button. The function begins by retrieving the values from the page using the document.getElementById() method and setting them as variables. These two variables are then parsed (converted) to integer values using the parseInt() method. This will ensure that the values are recognized as integers and not strings of numbers. Additionally, the Math.min() and Math.max() methods are used to create a minimum and maximum range of 0-100 to be displayed on click, regardless of the input values, as it is "THE HUNDRED" and not the "Infinity" app. Once these parameters are set, the function runs an if/else statement that executes the generateNumbers and displayNumbers functions, if and only if the input values are validated as integers. If the input is not validated as an integer and the user places a string in the value boxes, an alert will be generated instead.

GenerateNumbers

GenerateNumbers is the first of two logic functions. The function itself has two parameters the sValue and eValue. When this function is called in the controller function, getValues, it utilizes the two parameters by inputting the startValue and endValue in place of the sValue and eValue parameters. The function itself declares the variable "numbers and sets it to equal an empty array. This functions as an array constructor with an empty key. This array then uses a "for-loop" that sets the index of the array to the startValue, which will increment by one after each loop. The "for-loop" is set to execute as long as the index is less than or equal to the endValue and will continue to add new items to the end of the array using the push() method until that parameter is met. Once met, the generateNumbers function will return the array (numbers).

DisplayAlert

DisplayAlert is the second of two logic functions. This function is the only function of the four that behaves independently of the other 3. The purpose of this function is to ensure that the input values do not exceed 100 nor go below 0. Whereas the getValues function ensures that the displayed values can only generate values from 0-100, the displayAlert function ensures the values being inputted are not lower than 0 or larger than 100. If the user does input a value outside of these limits, an alert message will be generated and the value will be set to zero or 100 depending on the input field being used. This is accomplished using two if statements, the keyword "this" and the "value" property in conjunction.

DisplayNumbers

DisplayNumbers is the last function called in the initial function, getValues. This function works to display the values generated by the generateNumbers function to the viewport in a table. This is accomplished by utilizing a combination of a "for-loop", an if/else statement, and a template literal. The loop outputs a string, which will be the current value of the index of the number within the array of values generated. The loop continues this process for each index value within the array as long as the index remains less than the length of the array. The template literal is used to concatenate the empty string with the number value of the current index within the array. The if/else statement marks the even numbers in bold by using a modulo equation that sets their HTML class to 'even'. The function places this output in the table body of the results section by using the getElementById().innerHTML property-method.
